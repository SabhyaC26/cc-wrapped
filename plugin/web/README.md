# Claude Code Wrapped - Web Visualization

Web-based visualization component for Claude Code Wrapped statistics.

## Overview

This React application provides an interactive web interface to visualize your Claude Code Wrapped statistics with enhanced charts and animations.

## Development

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build

# Serve production build
bun run serve
```

## Data Format

The app expects a `public/data.json` file containing your Claude Code Wrapped statistics. This is automatically generated by the main plugin.

## Technology Stack

- React 19
- Custom build system with Bun
- CSS for styling
